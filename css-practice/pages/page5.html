Flexbox - Intro and Glowing & Shrinking

Flexbox is a way to manipulate elements in CSS

If another div is put into .flex-container, everything will "flex" to make it fit. Resizing the browser will also make the divs "flex" to fit.

A flex container is any element that has display: flex on it.
A flex item is any item that lives directly inside a flex container

In the above boxes: the yellow is the flex container, and the blue are the flex items

An element can be bother a container and an item

In the above boxes, the second box is both an item and a container, and so anything inside can be arranged with flexbox.
The third box is just an item, and so anything inside can't be arranged with flexbox.

flex is shorthand for flex: flex-grow flex-shrink flex-basis
When flex is defined with one value, it is applying to flex-grow

flex-grow expects a single number as it's value, which is the item's "growth factor".
If "1" is used, that means every item will grow to the same amount
If "2" is used on one div, it will grow 2x the size of the others (third boxes in examples above)

flex-shrink only ends up being applied if the size of all flex items is larger than their parent container.
The default is 1, so all items would shrink evenly.
Setting to 0 will mean the item won't shrink.
Setting it higher than 1 will mean the item will shrink higher than normal

If flex-grow or flex-shrink are specified, the items won't necessarily respect any given values for wdith. The default is to grow to fill the container, or shrink to fit it.

flex-basis is the initial size of a flex item, the default being 0% (but actually the default us auto). If auto is used, it tells the item to check for a declared width.

Most commonly, flex: 1; is used to make divs grow evenly, flex-shrink: 0 will keep certain divs from shrinking.
