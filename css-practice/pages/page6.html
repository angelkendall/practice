Flexbox - Axes

Flexbox is a way to manipulate elements in CSS

The default direction for a flex container is horiztonal (row).

It's possible to change the direction by using column:
flex-direction: column;

When using column, it's important to either:

  • use flex: 1 1 auto (instead of flex: 1)
  • putting a heitght in the parent .flex-container
  • using glex-grow: 1

Also, when using column, flex-basis refers to height instead of width
