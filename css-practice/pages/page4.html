Block and Inline

This page is about block and inline elements

block elements start on a new line, and will stack on top of each other by default.

inline elements don't start on a new line, and appear beside the elements they're placed next to.

Note: Generally don't want to put padding or margin on inline elements

Divs and spans are just boxes that can contain anything. Divs are block-level by default. Span is inline-level by default.
