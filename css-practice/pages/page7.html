Flexbox - Alignment

Adding flex: 1; to an item makes each item grow to fill the available space.

It's also possible to make the items stay the same/desired width and place them differently within a container.

justify-content aligns items across the main axis.

align-items places items along the cross axis

The above boxes are justified space-between, and aligned in the center.

If flex-direction is changed to column, justify-content and align-items are swapped:
justify-content will align vertically
align-items will align horiztonally

gap is a setting on a flex container that adds a specified space between flex items (similar to adding a margin).
This doesn't work when using justify-content:space-between;
