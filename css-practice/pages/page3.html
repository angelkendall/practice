The Box Model

This page will demonstrate the box model.

This is a box with a blue border. The padding is quite large, and there's a margin
Parts of the box:
  • Content box is where the content is displayed: width, height
  • Padding box sits around the content box: padding
  • Border box wraps with content and padding: border
  • Margin box is outside the others entirely: margin
width and height of a box will equal the content box. Padding and border is added to the width and height of the content box.

Note: The margin isn't counted toward the actual size of the box.
